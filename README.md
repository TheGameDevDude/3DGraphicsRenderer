# 3DGraphicsRenderer

+ 3D Rasterizer, made using java and uses no GPU.
+ Diffuse Lighting
+ Emissive mapping
+ Normal mapping
+ Parallax mapping

#### Keyboard
| Action | Keys |
|--------|------|
|Move Forward|<KBD>W</KBD>|
|Move Backwards|<KBD>S</KBD>|
|Move Left|<KBD>A</KBD>|
|Move Right|<KBD>D</KBD>|
|Move Up|<KBD>Space</KBD>|
|Move Down|<KBD>Shift</KBD>|
